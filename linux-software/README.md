# Software installation

Installs the following:

* vscode
* vscode extensions:
    * Ansible language support
    * Diff files
    * Rewrap
    * vscode-icons
    * YAML language support
    * Dracula theme
* customized `.bashrc`/`.zshrc` files

## System requirements

The playbook works for the following systems:

* RHEL 9.1+
* CentOS 8/9 Strema
* Ubuntu

## How-to

1) Run the `installer.sh`
1) Execute `run.sh`
